Documents embedded within.</p>
<p>The patents in question are:</p>
<ul>
<li><a target="_blank" href="http://www.google.com/patents/US5978791">Data processing system using substantially unique identifiers to identify data items, whereby identical data items.</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US6415280">Identifying and requesting data in network using identifiers which are based on contents of data</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US6928442">Enforcement and policing of licensed content using content-based identifiers</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US7802310">Controlling access to data in a data processing system</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US7949662">De-duplication of data in a data processing system</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US7945544">Similarity-based access control of data in a data processing system</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US7945539">Distributing and accessing data in a data processing system</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US8001096">Computer file system using content-dependent file identifiers</a></li>
<li><a target="_blank" href="http://www.google.com/patents/US8099420">Accessing data in a data processing system</a></li>
</ul>
