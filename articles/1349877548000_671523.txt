<p>I use Windows daily. I have since Windows 95. I wouldn&#8217;t call myself a fanboy, but I prefer Windows&#8217; desktop environment over that of OS X. Yet, like many of you, I&#8217;m not looking forward to Windows 8. These leaked Windows 8 ads found by an <a target="_blank" href="http://techit.co.il/2012/10/%D7%97%D7%A9%D7%99%D7%A4%D7%94-%D7%A4%D7%A8%D7%A1%D7%95%D7%9E%D7%95%D7%AA-%D7%94%D7%94%D7%A9%D7%A7%D7%94-%D7%A9%D7%9C-%D7%9E%D7%A2%D7%A8%D7%9B%D7%AA-%D7%94%D7%94%D7%A4%D7%A2%D7%9C%D7%94-windows-8/">Israeli website</a> show why.</p>
<p>Windows 8 is designed for the post-PC era, a time when we work and play on tablets, rather than archaic desktops. It&#8217;s designed to be operated by a finger rather than a mouse and keyboard. This philosophy shift is evident throughout the entire Windows 8 experience. Yet I&#8217;m tied to a desk with two 27-inch monitors for 12 hours a day. Windows 8 does not offer any advantage for the desktop user; the implementation of the classic Windows environment is a joke.</p>
<p>Windows 8 is a fantastic mobile OS, but my primary computer will not run Windows 8 until Steve Ballmer breaks down my door, subdues my crazy Yorkshire Terrier and installs it himself.</p>
