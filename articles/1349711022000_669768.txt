<img width="100" height="70" src="http://tctechcrunch2011.files.wordpress.com/2012/10/webplatform.jpg?w=100&amp;h=70&amp;crop=1" class="attachment-tc-carousel-river-thumb wp-post-image" alt="webplatform" title="webplatform" style="float: left; margin: 0 10px 7px 0;" /><p>A number of leading browser vendors and other tech companies, including <a target="_blank" href="http://blogs.msdn.com/b/interoperability/archive/2012/10/08/w3c-s-web-platform-docs-your-go-to-for-all-things-web-development.aspx">Microsoft</a>, <a target="_blank" href="http://blog.chromium.org/2012/10/teaming-up-on-web-platform-docs.html">Google</a>, Apple, <a target="_blank" href="http://blogs.adobe.com/digitalmedia/tag/web-platform/">Adobe</a>, Facebook, HP, Nokia, Mozilla, <a target="_blank" href="http://www.opera.com/press/releases/2012/10/08/">Opera</a> and the <a target="_blank" href="http://www.w3.org/2012/10/webplatform.html.en">W3C</a>, just announced the launch of the Web Platform Docs project at <a target="_blank" href="http://webplatform.org/">WebPlatform.org</a>. The project aims to create &#8220;a new, authoritative open web standards documentation site,&#8221; <a target="_blank" href="http://www.opera.com/press/releases/2012/10/08/">says Opera Software</a>. The wiki-like site, says Opera, wants to ensure that developers can easily find &#8220;accurate, quality information on all the latest HTML5, CSS4 and other standards features across the multitude of available web-based resources.&#8221;</p>
<p><a href="http://techcrunch.com/2012/10/08/google-microsoft-facebook-and-others-launch-web-platform-docs-a-web-standards-documentation-site/css-color-color-table/" rel="attachment wp-att-669797"></a>Currently, the companies behind WebPlatform.org <a target="_blank" href="http://blogs.adobe.com/digitalmedia/tag/web-platform/">argue</a>, developers struggle to find authoritative answers to their questions about modern web technologies and often, developers have to resort to figuring out the right solutions through trial and error (the Google team <a target="_blank" href="http://blog.chromium.org/2012/10/teaming-up-on-web-platform-docs.html">describes this</a> as a &#8220;scavenger hunt&#8221;). The new site, says Adobe, will change this by providing developers a &#8220;single, definitive resource to go to.&#8221; On the site, users will find API documentation, information on browser compatibility, examples, best practices and the status of the various specifications.</p>
<p>The site has been seeded with information from the participating organizations, but anyone will be able to contribute content to the project. The W3C will serve as the site&#8217;s convener and curator, but the various participating organizations stress that this is a community effort.</p>
<p>&#8220;People in the web community — including browser makers, authoring tool makers, and leading-edge developers and designers — have tremendous experience and practical knowledge about the web,&#8221; <a target="_blank" href="http://www.w3.org/2012/10/webplatform.html.en">said</a> W3C Director Tim Berners-Lee in a canned statement today. &#8220;Web Platform Docs is an ambitious project where all of us who are passionate about the web can share knowledge and help one another.&#8221;</p>
