<img width="100" height="70" src="http://tctechcrunch2011.files.wordpress.com/2012/08/chrome-logo.jpg?w=100&amp;h=70&amp;crop=1" class="attachment-tc-carousel-river-thumb wp-post-image" alt="chrome-+-logo" title="chrome-+-logo" style="float: left; margin: 0 10px 7px 0;" /><p>Google just <a target="_blank" href="http://chrome.blogspot.com/2012/09/chrome-for-android-just-got-safer.html">launched</a> an update to <a target="_blank" href="https://play.google.com/store/apps/details?id=com.android.chrome">Chrome for Android</a> that provides the browser with a strengthened sandbox. This, says Google, will help ensure that malicious mobile sites remain contained and can&#8217;t impact the enture browser. It&#8217;s worth noting, though, that this enhanced sandbox will only be available to Android 4.1 Jellybean users. All Chrome for Android users (Chrome is only available for Android 4.0+), though, will profit from the <a target="_blank" href="http://googlechromereleases.blogspot.com/2012/09/chrome-for-android-update.html">additional security fixes</a> and small feature updates in this release. In total, Google paid out $750 to developers who found security issues.</p>
<p>As Google notes, the Chrome sandbox &#8220;is made possible by the innovative multi-process architecture in Chrome for Android, in conjunction with Android’s <a target="_blank" href="http://source.android.com/tech/security/#the-application-sandbox">User ID (UID) isolation</a> technology.&#8221;</p>
<p>Besides these security fixes, this update also makes a few small changes to how YouTube videos work in Chrome. YouTube video controls, Google says, now work in full-screen mode and video now automatically continue playing after a screen lock/unlock. Google also says that third-party input methods (think <a target="_blank" href="http://www.swiftkey.net/">SwiftKey</a> or <a target="_blank" href="http://www.swype.com/">Swype</a>) will now work better with Chrome.</p>
<p>While the stronger sandbox is limited to Jellybean, all of the other fixes and feature updates will be available for all Android 4.0+ users.</p>
